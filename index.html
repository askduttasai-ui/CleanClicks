Security

Built to be trusted.

7 protection layers. CleanClicks never touches the internet after install.

01
Localhost-Only API
All requests from external IPs blocked with HTTP 403. No hacker can reach your API.
02
Security Headers
XSS, clickjacking, MIME sniffing protection on every response. CSP enforced.
03
CORS Locked
Only localhost:5050 can call the API. All other origins permanently blocked.
04
SHA-256 Integrity
Checks its own file hash on every startup. Warns immediately if tampered with.
05
Session Tokens
256-bit random token generated fresh every startup. Expires when app closes.
06
Zero Internet
No telemetry, analytics, update checks, or cloud sync. Everything stays on your machine.
07
Open Source
Every line of code publicly auditable on GitHub. Nothing hidden. Nothing to hide.
🔍CleanClicks never reads the content of your files — only paths, sizes, and dates. Your documents and photos are never accessed.
